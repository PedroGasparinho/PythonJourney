\begin{frame}
    \frametitle{Tipos de dados em Python}
    \begin{itemize}
        \item Similarmente à Matemática, em Python e noutras linguagens de
            Programação, existem conjuntos de valores, denominados tipos de dados.
        \item No entanto, estes não se limitam apenas a números, existem também 
            formas de representar texto, verdadeiros, falsos, entre outros.
        \item Um valor pode ter diferentes representações, mas cada representação
            apenas está associada a um tipo de dados. Por exemplo, 1 é um valor
            inteiro, enquanto 1.0 é considerado real.
    \end{itemize}
\end{frame}

\begin{frame}
    \frametitle{Tipos de dados em Python}
    \begin{itemize}
        \item Números Inteiros (Integers)
            \begin{itemize}
                \item Denominado \textit{int}
                \item Exemplos: \textit{1}, \textit{-4}, \textit{0}
            \end{itemize}
        \item Números Reais (Floats)
            \begin{itemize}
                \item Denominado \textit{float}
                \item Exemplos: \textit{1.0}, \textit{-6.2}, \textit{3.1415926}
            \end{itemize}
        \item Valores lógicos (Booleans)
        \begin{itemize}
            \item Denominado \textit{bool}
            \item Exemplos: \textit{True}, \textit{False}
        \end{itemize}
    \end{itemize}
\end{frame}

\begin{frame}
    \frametitle{Tipos de dados em Python}
    \begin{itemize}
        \item Texto (Strings)
        \begin{itemize}
            \item Denominado \textit{str}
            \item Exemplos: \textit{'1'}, \textit{"Hello, world"}, \textit{"""Olá"""}
        \end{itemize}
    \item Listas
        \begin{itemize}
            \item Denominado \textit{list}
            \item Exemplos: \textit{[1, 2, 3]}, \textit{[]}, \textit{["Olá", "Olé"]}, \textit{["Olá", 1, ["ok"], true]}
        \end{itemize}
    \item Entre outros...
    \end{itemize}
\end{frame}

\begin{frame}
    \frametitle{Algumas notas sobre Floats}
    \begin{itemize}
        \item Não é possível representar fidedignamente todos os números 
        reais, como por exemplo, os números com dízimas infinitas, já que os 
        computadores têm espaço de armazenamento finito.
        \item Outro exemplo clássico é somar \textit{0.1} com \textit{0.2}, que 
        resulta em \textit{0.30000000000000004}.
        \item Isto deve-se à norma IEE 754, usada para representar números reais,
        mas que não será discutida nestes materiais.
    \end{itemize}
\end{frame}

\begin{frame}
    \frametitle{Algumas notas sobre Strings}
    \begin{itemize}
        \item Devem começar com um apóstrofe, um par de aspas ou 
        três pares de aspas e terminar com o mesmo caracter.
        \item Existem diferenças entre as 3 possibilidades, mas apenas serão 
            discutidas mais tarde.
    \end{itemize}
\end{frame}

\begin{frame}
