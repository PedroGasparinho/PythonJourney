\subsection*{Exercise 3 - List Summation}

Let $a = a_{0}, ..., a_{n-1}$, then $a$ is sequence with $n$ elements, starting from
index $0$ to $n-1$. The sum of the elements in sequence $a$, can be expressed as: 

\[ \sum_{i=0}^{n-1} a_{i} = a_{0} + ... + a_{n-1} \]

\[\text{Digital root of } 9045 = 9 \text{, because } 9+0+4+5 = 18 \text{ and } 
1+8 = 9 \]

\subsection*{Exercise 16 - Palindromes}
A palindrome is a sequence of symbols that can be read forward and backwards
in the same way. However, one may usually find that some liberties are taken,
for instance with punctuation or formatting. For simplicity, we recommend
only considering truly palindromic sequences of characters, i.e. the characters
must exactly match in both ways, with the only exception being the letter cases, 
if the student wishes. \\

\noindent By our definition:
\noindent \begin{itemize}
  \item "Stop pots" is truly palindromic, if one ignores letter cases;
  \item "Never odd or even" is not truly palindromic, since the 5th character 
    from the start is "r" and from the back is a space.
\end{itemize}

\begin{enumerate}[a)]
  \item Develop a function that determines whether a string is truly palindromic.
  \item Develop a test function for the Palindromes problem.
\end{enumerate}

\subsection*{Exercise 22 - Matrix Sum}
Let $A$ and $B$, be two $n*n$ matrices, then:
\[ A+B = 
  \begin{bmatrix} 
    a_{11} & ... & a_{1n} \\ 
    ... & ... & ... \\
    a_{n1} & ... & a_{nn} 
  \end{bmatrix} + 
  \begin{bmatrix} 
    b_{11} & ... & b_{1n} \\ 
    ... & ... & ... \\
    b_{n1} & ... & b_{nn} 
  \end{bmatrix} =
  \begin{bmatrix} 
    a_{11} + b_{11}  & ... & a_{1n} + b_{1n} \\ 
    ... & ... & ... \\
    a_{n1} + b_{n1}  & ... & a_{nn} + b_{nn} 
  \end{bmatrix}  
\]

\noindent For instance:
\[
  \begin{bmatrix} 
    1 & 2 & 3 \\ 
    4 & 5 & 6 \\
    7 & 8 & 9 
  \end{bmatrix} + 
  \begin{bmatrix} 
    1 & 0 & 0 \\ 
    0 & 1 & 0 \\
    0 & 0 & 1 
  \end{bmatrix} =
  \begin{bmatrix} 
    2 & 2 & 3 \\ 
    4 & 6 & 6 \\
    7 & 8 & 10 
  \end{bmatrix}
\]
\begin{enumerate}[a)]
  \item Develop a function to check if the matrix has the same number of lines
    and columns.
  \item Develop a test function for the previous subproblem
  \item Develop a function that calculates the sum of two matrices. 
  \textbf{Suggestions:} Guarantee that both matrices are $n*n$ and the result 
    should be a new matrix.
  \item Develop a test function for the Matrix Sum problem.
\end{enumerate}

  \item Develop a function that calculates the transpose of a matrix. 
  \textbf{Suggestions:} Guarantee that it is a valid matrix and the result 
    should be a new matrix.
  \item Develop a test function for the Matrix Transposition problem.
\end{enumerate}

\subsection*{Exercise 25 - Sorted Concatenation}
Concatenation commonly refers to the operation of joining two lists, usually
the second list is placed at the end of the first. In Python this can be 
achieved with the 'extend' operation. 

\begin{pythonlst}
l1 = [1, 4, 5, 7]
l2 = [2, 3, 6]
l1.extend(l2)
# l1 = [1, 4, 5, 7, 2, 3, 6]

l3 = [1, 2, 3, 4]
l4 = [5, 6, 7]
l3.extend(l4)
# l4 = [1, 2, 3, 4, 5, 6, 7]
\end{pythonlst}

\noindent However, by naively concatenating two sorted lists the resulting list 
may lose that property (unless every element of $l1$ is lesser or equal that 
every element of $l2$). One possible solution would be to naively concatenate 
the two list then sort the result, which is not an interesting exercise, since 
Python offers a built-in function for sorting lists. Instead, we propose to 
build a new list (unlike 'extend' which alters the first list) and insert the 
element in order, for instance, for $l1$ and $l2$:

\begin{figure}[h]
  \includegraphics[width=0.70\textwidth]{../Images/SortedConcat.png}
  \centering
\end{figure}

\noindent \textbf{Suggestion:} To achieve this, use a while loop that iterates 
over both lists at the same time, but only advance one element of one of the 
lists at a time.

\begin{enumerate}[a)]
  \item Develop a function that performs the sorted concatenation of two lists. 
  \item Develop a test function for the Sorted Concatenation problem.
\end{enumerate}
